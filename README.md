# CSCE-4350.401-Build-a-Database-Project

This python program utilizes an array to store keys and values using GET, SET, and EXIT commands. All changes are persisted to data.db creating consistent data storage that rebuilds upon restarting the program. 

The SET command updates stored memory and properly updates or appends data depending on whether the key exists or not. The GET operation performs a linear search in the stored values and properly returns assignments including NULL when the key does not exist. The EXIT command cleanly terminates the while loop when prompted.
